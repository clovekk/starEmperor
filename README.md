# :information_source: Game description
Star Emperor is a simple 2D strategic game about conquest of star systems in the galaxy. 
The premise is easy to understand, you start with controlling one system and one star ship fleet, your goal is to control as many star systems as possible while your resource stockpile stays positive. 

# :joystick: Gameplay
You can see information about your stockpile on the top bar that contains all the various buttons important for gameplay. 
By clicking the resource buttons you can see where you're getting them from and what consumes them. 
By clicking on systems you can open the system info window that contains information about the systems's resoruces and buttons to send the selected to the system or to claim the system.
You can save your game and load it later whenever you wish, while you're in-game you can pause the game as well.
